# California Housing Price Prediction

This project predicts California housing prices using machine learning methods in Python.

## Contents
- `California_Housing_Project.ipynb`: main notebook containing data preprocessing, model training, and evaluation

## Objective
The goal of this project is to analyze housing-related features and build models to predict house prices.

## Tools
- Python
- Jupyter Notebook
- pandas
- numpy
- scikit-learn
- matplotlib / seaborn

## Workflow
1. Load and explore the dataset
2. Clean and preprocess data
3. Train machine learning models
4. Evaluate model performance
5. Compare results and draw conclusions

## Result
The project compares different regression approaches and identifies the model with the best predictive performance.
